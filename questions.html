1. Expliquer la différence entre une requête HTTP synchrone et asynchrone.

La différence fondamentale entre une requête synchrone et asychrone c'est que la requête dite synchrone bloque les envois de requêtes en attendant la réponse, tandis que l'asynchrone peut envoyer plusieurs requêtes sans avoir besoin d'attendre la réponse.
Fun fact : Le premier système largement utilisé pour des requêtes asycrones sur le web s'appelait AJAX

3. Écrire vos recommandations (best practice) pour programmer un site web en HTML / CSS.

Commencer par se documenter et construire la structure du site sur papier, choisir les liens entre les pages et leur disposition.
Une fois fait, choissisez une ligne éditoriale, qu'elle est sont but, qu'elle palette de couleur choisir, l'accès facile a l'information etc.
Ensuite une fois devant votre code commencer par crée votre site pour le mobile et l'adapter ensuite pour un écran plus large.
Et pour facilité la conception du site je proposerais de prendre des libraire comme bootstrap par exemple.
